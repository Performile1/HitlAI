## üéØ What is HitlAI?

HitlAI is a **hybrid AI-human UX testing platform** that combines artificial intelligence with real human testers to provide comprehensive usability testing for websites and mobile applications.

### **Core Concept**

Traditional UX testing is either:
- **Too slow**: Manual testing takes weeks
- **Too expensive**: Professional testers cost $50-100/hour
- **Too limited**: AI-only testing misses human nuances

**HitlAI solves this** by:
1. AI agents test first (fast, cheap baseline)
2. Human testers validate (authentic feedback)
3. AI learns from humans (improves over time)

---

## üèóÔ∏è How It Works

### **For Companies**

1. **Create Test Request**
   - Upload app (.apk, .ipa) or provide URL
   - Define test objectives
   - Select target demographics (age, tech literacy, disabilities)
   - Choose AI/Human ratio (e.g., 70% AI, 30% Human)
   - Purchase credits

2. **AI Tests First**
   - 12 specialized AI agents execute automatically
   - Crawl website with JavaScript rendering
   - Identify friction points
   - Generate baseline report
   - Flag areas needing human validation

3. **Human Testers Validate**
   - Real users test the app/site
   - Follow AI-suggested paths
   - Add comments and annotations
   - Confirm or dispute AI findings

4. **Receive Combined Report**
   - AI + Human insights merged
   - Friction points with severity ratings
   - Video recordings of human tests
   - Actionable recommendations

### **For Human Testers**

1. **Sign Up & Profile**
   - Create profile with demographics
   - Set availability and rates
   - Complete qualification tests

2. **Accept Tests**
   - Browse available tests
   - Accept tests matching profile
   - View test objectives

3. **Execute Test**
   - View app/site in test viewer
   - Follow test instructions
   - Add comments and annotations
   - Submit results

4. **Get Paid**
   - Earn $5-15 per test
   - Performance-based pay (higher quality = higher rates)
   - Weekly payouts

## ü§ñ AI Agent System

### **12 Specialized Agents**

1. **MissionPlanner**: Creates test strategy and step-by-step plan
2. **VisionSpecialist**: Audits UX with persona-weighted heuristics
3. **TechnicalExecutor**: Generates and executes Playwright scripts
4. **CritiqueAgent**: Validates AI findings against human feedback
5. **GlobalInsightsAgent**: Strategic analysis and systemic gap identification
6. **PersonaFromTesterAgent**: Learns from human tester behavior
7. **MemoryManager**: Stores and retrieves lessons from past tests
8. **FrustrationMeter**: Detects user frustration patterns
9. **CrossVerifier**: Prevents hallucinations in vision analysis
10. **TesterVerifier**: Detects AI-generated fake reports
11. **VelocityChecker**: Identifies bot testers
12. **ScreenshotAnonymizer**: Removes PII from screenshots

### **Agent Orchestration**

- **Orchestrator**: Manages agent execution flow
- **AgentMonitor**: Prevents zombie processes, enforces timeouts
- **CircuitBreaker**: Budget protection, prevents runaway costs
- **ContextPruner**: Token management, reduces context bloat
